# AI Fortune - AI算命应用规格文档

## 1. 项目概述
- **项目名称**: AI Fortune (AI算命)
- **类型**: Android 原生应用
- **核心功能**: 集成多种传统命理分析工具，支持用户自定义大模型API

## 2. 技术栈
- **语言**: Kotlin 2.0
- **UI框架**: Jetpack Compose + Material 3
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **网络**: Retrofit2 + OkHttp
- **存储**: DataStore (偏好设置)
- **动画**: Compose Animation APIs

## 3. 界面结构

### 3.1 底部导航 (Bottom Navigation)
- 首页 (Home)
- 功能 (Features)  
- API设置 (API Settings)
- 我的 (Profile)

### 3.2 页面列表
1. **首页** - 欢迎页，展示主要功能入口
2. **功能页** - 各个算命功能的列表
3. **八字命理页** - 输入出生信息，分析命理
4. **求学建议页** - 输入信息，获取学业建议
5. **求商建议页** - 输入信息，获取商业决策建议
6. **星座分析页** - 输入星座，获取运势分析
7. **姓名分析页** - 输入姓名，分析含义
8. **塔罗牌页** - 抽牌并解读
9. **起名字页** - 根据条件生成名字建议
10. **API设置页** - 配置大模型API
11. **我的页面** - 历史记录、设置

## 4. 功能详情

### 4.1 八字命理
- 输入：姓名、出生日期（年/月/日/时）、性别
- 输出：命理分析、大运流年、职业建议

### 4.2 求学建议
- 输入：姓名、年级、想报考的学校/专业
- 输出：学业规划、复习建议、报考策略

### 4.3 求商建议
- 输入：想做的事情、预算、风险偏好
- 输出：商业分析、风险评估、时机建议

### 4.4 星座分析
- 输入：星座、问题类型（事业/爱情/财运/健康）
- 输出：今日/周/月运势

### 4.5 姓名分析
- 输入：姓名
- 输出：姓名含义、五行分析、运势加成

### 4.6 塔罗牌
- 选择牌阵（单张/三张/马蹄铁）
- 随机抽牌
- AI解读

### 4.7 起名字
- 输入：姓氏、性别、期望含义
- AI生成名字建议

### 4.8 API设置
- 支持添加多个API配置
- 每个配置包含：
  - API名称（显示用）
  - API URL（必需）
  - 模型名称（必需）
  - API Key（可选）
  - 最大Token数（默认2048）
  - Temperature（默认0.7）
  - 系统提示词（可选）
- 支持设为默认API
- 支持测试连接

## 5. UI/UX 设计

### 5.1 配色方案
- **主色**: #6366F1 (Indigo 500)
- **次色**: #8B5CF6 (Violet 500)
- **背景**: #0F0F23 (深色) / #FAFAFA (浅色)
- **卡片**: #1A1A2E (深色卡片) / #FFFFFF (浅色卡片)
- **文字**: #E2E8F0 (深色) / #1E293B (浅色)
- **强调色**: #F59E0B (琥珀色，用于吉祥元素)

### 5.2 动画要求
- 页面切换：共享元素过渡 + 淡入淡出
- 卡片加载：交错展开动画
- 按钮点击：水波纹 + 缩放反馈
- 抽卡动画：3D旋转效果
- 结果展示：打字机效果 + 渐显

### 5.3 字体
- 标题：思源黑体 / Noto Sans SC (Bold)
- 正文：思源黑体 / Noto Sans SC (Regular)

## 6. 数据流

```
User Input → ViewModel → Repository → API Service → LLM API
                ↓
         Result Screen (Compose UI)
```

## 7. 存储
- **DataStore**: 存储用户API配置、历史记录、偏好设置
- **JSON序列化**: Gson

- 网络超时提示
- API返回格式错误提示
